# ptfl-comments

**The public discussion backend for [tokarevxvi.dev](https://tokarevxvi.dev)**

This repository powers the comment system on my blog via [Giscus](https://giscus.app) + GitHub Discussions. Every blog post has a corresponding discussion thread here.

## Why a separate repo?

It's clean architecture — discussions live where they belong, not buried in source code.

## How it works

- Comment on a post → new Discussion created here (auto-mapped by pathname)
- Discussions are public and readable
- All you need is a GitHub account
